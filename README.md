# SMS-Backup-Explorer
View and browse SMS backups generated by SMS Backup &amp; Restore app - Free and Private

A lightweight, browser‑based viewer for XML files from **SMS Backup & Restore** app.  
Designed to make exploring your SMS history fast, intuitive, and pleasant — with a clean chat‑style interface, powerful filters, and full dark/light mode support.

---

## 💡 Why This App Is Useful

Managing SMS backups in XML format is surprisingly difficult. The XML files generated by **SMS Backup & Restore** contain all your messages, but they’re not meant to be read directly and easily. Opening them in a text editor is overwhelming, and general XML viewers are not optimized for this backup. This web app solves that problem by turning raw XML into a clean, modern, chat‑style interface that feels familiar and easy to navigate.

It’s especially helpful when you need to **search through years of conversations**, **find a specific message**, or **review communication history** for personal, legal, or sentimental reasons. Instead of scrolling through thousands of lines of XML, you can filter by contact, date, message type, or even search inside message bodies — all instantly, directly in your browser.

Because everything runs **client‑side**, your messages never leave your device. This makes the app ideal for anyone who wants convenience without sacrificing privacy. Whether you're restoring old memories, verifying information, or simply organizing your backups, this tool gives you clarity and control over your SMS history.

---

## 🌟 Features

- **Chat‑style message view** — Messages are displayed like a modern messaging app  
- **Contact filtering** — Filter by contact name or number, with instant search  
- **Date range filtering** — View messages from any specific time window  
- **Message type filtering** — Show only *Sent*, *Received*, or both  
- **Full‑text search** — Search inside message bodies  
- **Sorting options** — Sort messages newest→oldest or oldest→newest  
- **Dark & Light themes** — Automatic or manual theme switching  
- **Runs entirely in your browser** — No uploads, no servers, your data stays local

## 🚀 Getting Started

1. Export your SMS backup using **SMS Backup & Restore** (XML format).
2. Open this web app (You can download and run the app locally in your browser or simply go to [project GitHub Page](https://amirhosainshahsavari.github.io/SMS-Backup-Explorer/)).
3. Load the XML file — messages appear instantly in a chat‑style layout.
4. Use filters, search, and sorting to explore your conversations.
